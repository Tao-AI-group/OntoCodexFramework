# OntoCodexFramework
OntoCodex is a multi-agent framework that integrates multiple agent to use ontology based knowledge retrieval and reasoning for enhancment of biomedical ontologies. 

OntoCodex GPT is created based on this framework. It can be accessed from ***https://chatgpt.com/g/g-6734e1cd43008190b7245d756c8e75ef-ontocodex
